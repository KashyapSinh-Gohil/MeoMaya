Clean Pipeline

Normalize → Tokenize → Tag → Parse. Modular components, easy to understand and extend.

Local‑First

Runs fully on‑device. Optional HF models are strictly local (no downloads).

Multimodal (Optional)

Image, audio, and video pipelines extract real metadata when local libs are available.

Hardware‑Aware

Detects CPU/CUDA/MPS without forcing heavy dependencies in the core.

Developer Friendly

Simple API, FastAPI server, CLI, thorough tests, and readable code.

Offline Mode

Set MEOMAYA_STRICT_OFFLINE=1 to disable downloads and use lightweight fallbacks.

Tiny, powerful building blocks

Tokenizer

from meomaya.core.tokenizer import Tokenizer

print(Tokenizer().tokenize("Hello, world!"))

Tagger

from meomaya.core.tagger import Tagger

tagger = Tagger()
print(tagger.tag(["This","is","MeoMaya"]))

Parser

from meomaya.core.parser import Parser

print(Parser().parse([("MeoMaya","NN")]))

Getting Started

  1. Clone the repository and create a virtual environment.
  2. Install runtime dependencies.
  3. Run the API or the CLI. Explore the documentation for more.
git clone https://github.com/KashyapSinh-Gohil/MeoMaya.git
cd MeoMaya
python -m venv .venv
source .venv/bin/activate
pip install -r meomaya/requirements.txt

# Optional dev/API tooling
pip install -r requirements-dev.txt
